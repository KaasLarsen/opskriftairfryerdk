## Månedlig GSC-rutine

1. Eksporter **effektivitet → forespørgsler** (eller sammenligning) for property `https://www.opskrift-airfryer.dk/` med samme CSV-struktur.
2. Opdatér denne mapping for de **øverste 50** hver gang, eller tilføj nye rækker nederst med dato.
3. Prioritér: **klik > 20** eller **eksponeringer > 500** med **placering 4–15** → `optimer` først (hurtig gevinst).
4. Efter publicering af nye eller opdaterede opskrifter: notér dato ved slug; efter 4–8 uger tjek samme forespørgsel for **CTR** og **gennemsnitlig placering** i GSC.
